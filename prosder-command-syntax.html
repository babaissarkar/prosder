Prosder syntax help

Subhraman Sarkar, 05.08.2020

 

Configuration commands :

CLR

Clears the screen. Also available from Draw menu -> Clear screen.

GRID ON/GRID OFF

Toggles the drawing of the grid and axes.

COL <R> <G> <B>

Set the current foreground color to the color corresponding to the given RGB value.  All three values (<R>, <G>, <B>) are integers and run from 0 to 255. For example, COL 255 0 0 sets the foreground color to Red, while COL 255 255 0 sets it to Yellow. A graphical color selection dialog is available via the Draw Menu -> Colors...

LW <width>

Sets the width of the line drawn. <width> is given in pixels and is a floating point value. i.e., LW 5.0 sets the current stoke width to 5.0px.

Drawing commands :

MV <X> <Y>

Move the current position of the drawing cursor to the point (<X>,<Y>). Note that the origin is at the middle of the drawing area of the screen.

MVR <DX> <DY>

If the current coordinate of the drawing cursor is (curX, curY), then move the drawing cursor to the point (curX+DX, curY+DY).

LN <X> <Y>

Draw a line from the current position of the cursor to the point (<X>,<Y>) using the present Stroke Width and Foreground color. Also sets the current position of the drawing cursor to (<X>,<Y>).

ARC <CX> <CY> <RAD> <ANG1> <ANG2>

Draw the segment of the circle with center at (<CX>, <CY>) and radius <RAD> enclosed between the angles <ANG1> and <ANG2> in the counter-clockwise sense.

Looping command :

LOOP <N>

...

LEND

Any command written between the LOOP and LEND <N> statements is repeated <N> times.

Transformation commands :

TRN <DX> <DY>

Translate the origin and the axes by <DX> units along the X axis and <DY> units along the Y axis.

ROT <ANG>

Rotate the axes <ANG> degrees counterclockwise.
